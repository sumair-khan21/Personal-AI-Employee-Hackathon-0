Here is what I built in Silver Tier:

Gmail Watcher
Monitors my inbox every 2 minutes. Detects important emails, classifies urgency, and creates structured action files automatically. No more missed replies.

AI-Drafted Email Replies
Claude reads the email, writes a professional reply, and waits for my approval before sending a single word. Full human-in-the-loop control.

File Drop Vault
Drop any document — invoice, PDF, CSV — into a folder. The AI detects it, summarizes it, and routes it to the right place. Built on Obsidian so everything stays local and searchable.

Monday CEO Briefing
Every Monday morning, a briefing is auto-generated covering pending tasks, completed work, approvals waiting, and priorities for the week.

Human-in-the-Loop (HITL)
Every sensitive action — emails, posts, payments — requires explicit human approval before execution. Claude creates an approval request and stops. You decide. No surprises.

Tech Stack
- Claude Code (reasoning engine)
- Python and Gmail API (email automation)
- Playwright (browser automation)
- Obsidian (local AI dashboard)
- Cron (fully scheduled, runs 24/7)
